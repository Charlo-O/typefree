# ChatGPT Codex Landing Reference

This folder is a local reference capture for rebuilding the visual structure of
`https://chatgpt.com/zh-Hans-CN/codex/`.

The downloaded images are third-party OpenAI/ChatGPT assets and should remain
reference-only. Do not ship them in the TypeFree production bundle. Use the
static `index.html` page here to compare section rhythm, spacing, card shapes,
image scale, and responsive behavior before replacing the assets and copy with
TypeFree-owned material.
